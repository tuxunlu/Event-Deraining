This repo consists of event based deraining experiments conducted for CMSC848B Computational Imaging class taken on Fall '25 at the University of Maryland- College Park, MD. 
Project members: Tuxun Lu, Ishan Tamrakar 

## Image + Video Datasets
### [Snow100k](https://drive.google.com/file/d/1gLE6hYyIiQypV5dKPNlNkzr9JBgpXdP2/view?usp=drive_link) 
Image dataset with synthetic snow (photoshop) 
[paper](https://arxiv.org/pdf/1708.04512)

### [SnowCityscapes](https://drive.google.com/file/d/1HmxhQiZa8XOH1IFB76htAQLV8z26aM0O/view?usp=drive_link)
2k+2k images
[paper](https://arxiv.org/pdf/2103.11298)

### [SnowKitti2012](https://drive.google.com/file/d/1-34ruS8KpLOU-srYCsOcZO0YuF-ELXwc/view?usp=drive_link) 
[paper](https://arxiv.org/pdf/2103.11298)

### [SnowVideo](https://drive.google.com/file/d/18WbSmEwXsbsrAVbAN8uPZYUXtyAHjBsq/view?usp=drive_link)
Video dataset. 110 videos 90k frames
[paper](https://openaccess.thecvf.com/content/ICCV2023/papers/Chen_Snow_Removal_in_Video_A_New_Dataset_and_A_Novel_ICCV_2023_paper.pdf)
